# GCP to Azure Pipeline Migration

This project demonstrates migrating a GCP pipeline to Azure using Terraform.

## Technologies Used
- Azure
- Terraform
- GitHub

## Setup
1. Install Azure CLI
2. Install Terraform
3. Run `az login`
4. Create Service Principal
5. Update `terraform.tfvars` with credentials
6. Run `terraform init`
7. Run `terraform plan`
8. Run `terraform apply`
